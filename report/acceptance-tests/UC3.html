UC3 - Submit new Receipt
Data:
  • <ReceiptID>
  • <Date>
  • <Kind>
  • <Amount>
  • <Company>
  • <Country>
  • <City>
  • <Street>
  • <Number>
Primary Course:
  1. User issues `Add Receipt` command.
  2. User picks all data for the creation of a new receipt.
  3. User confirms and provides data.
Expected Result:
  1. User is asked a receipt id, a date kind and amount of receipt as well as a company name, country, city, street, and number for creating a receipt.
  2. -
  3. System adds the receipt in memory and updates the log files accordingly.
