UC1 - Load Taxpayer Information from File
Data:
  • <Taxpayer AFM>
Main Flow:
  1. User issues `Load Taxpayer` command.
  2. System uses the AFM provided to find the corresponding file.
  3. System parses data and stores it in memory.
Non Existent AFM (2)
  1. User is notified with: `The file doesn't exist.`.
  2. UC1 is executed again.
Invalid AFM (2)
  1. User is notified with: `The tax registration number must have 9 digits. Try again.`.
  2. UC1 is executed again.
