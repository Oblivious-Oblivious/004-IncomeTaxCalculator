UC2 - Select and Display Taxpayer Information
Data:
  • <Taxpayer AFM>
Main Flow:
  1. User issues `Select Taxpayer` command.
  2. System uses provided AFM to search and display from the taxpayers loaded.
  3. System displays taxpayer data about personal information and receipts.
Non Existent AFM (2)
  1. User is notified with: `This tax registration number isn't loaded.`.
  2. Use case ends.
Invalid AFM Error (2)
  1. User is notified with: `You must give a tax registration number.`.
  2. Use case ends.
